django-polls
============

A Django app to conduct web-based polls.

Installation
------------

.. code-block:: bash

    pip install django-polls

Usage
-----

Include the app in INSTALLED_APPS:

.. code-block:: python

    INSTALLED_APPS = [
        ...,
        "django_polls",
    ]

Links to Django
---------------

You can see Django classes like :class:`django.test.TransactionTestCase`.
